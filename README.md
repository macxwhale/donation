# 1,000 Bibles for Kenya

A modern, responsive one-page donation website for crowdfunding 1,000 Bibles for underprivileged children in Kenya.

## Features
- **Emotional AI-Generated Imagery**: Culturally authentic visuals showing the impact of the mission.
- **Dynamic Progress Bar**: Real-time animation showing campaign progress.
- **Responsive Design**: Optimized for all devices using CSS Grid and Flexbox.
- **Render Ready**: Includes `render.yaml` for instant deployment.

## Tech Stack
- HTML5
- Vanilla CSS3
- JavaScript (Vanilla)

## Deployment
This site is configured for [Render](https://render.com). Simply connect your GitHub repository and it will deploy automatically.
